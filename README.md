# ethergas-api
Ethergas oracle api

Requirements:
 - nodejs v10+
 - python3
 - pip3

Installation:
 - sudo apt install python3-pip
 - pip3 install -r required.txt
 - yarn/npm install
 - create a config.json using the config.template.json file
 - set workers to false if you intend to use it as an api route

start:
 - yarn/npm start
